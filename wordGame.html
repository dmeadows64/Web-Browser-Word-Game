A Simple Word Game

  • In this game you will be given five random letters. Combine these letters together to make words.
  • All words are a maximum of five letters long. Individual words may only be used once.
  • Each letter can only be used once per word, but you do not have to use all five letters.
  • Click the Submit button if you think you have a word. If it is on the list (consisting of 1194 common English words) then it will appear on the list below.
  • You will get a number of points equal to the number of letters in the word you submit.
  • If you cannot think of a word with the letters given then click the Randomize button to generate five new letters.
  • After starting click on the Quit button to end the game and bring this text back.
